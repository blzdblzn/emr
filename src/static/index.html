API Documentation

RESTful API Endpoints

The system provides comprehensive API endpoints for all modules:

  • /api/auth - Authentication and user management
  • /api/patients - Patient information management
  • /api/medical-records - Medical record operations
  • /api/appointments - Appointment scheduling
  • /api/billing - Billing and payment processing
  • /api/insurance - Insurance information management
  • /api/hmo - HMO provider and contract management
  • /api/claims - Claims submission and tracking
  • /api/reconciliation - Claim reconciliation operations
  • /api/reports - Reporting and analytics

For detailed API documentation, please refer to the API guide.
